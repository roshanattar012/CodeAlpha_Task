## 📌 Project Overview
This project demonstrates web scraping using Python and BeautifulSoup. The scraper extracts product details such as title, price, and ratings from a website and stores them in a structured dataset for analysis.

---

## 🎯 Objectives
✅ Automate data collection  
✅ Extract product information from websites  
✅ Generate structured datasets  
✅ Analyze scraped data  
✅ Visualize product rating distributions  

---

## 🛠️ Technologies Used
- Python 🐍
- Requests
- BeautifulSoup
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
